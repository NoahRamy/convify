1. How Convify works

Convify is a client-side web application. All file conversion processing happens entirely within your web browser using JavaScript. When you select a file to convert:

  • The file is read locally by your browser
  • The conversion is performed entirely on your device
  • The converted file is made available for you to download locally
  • No file data is ever transmitted to our servers or any third-party servers

2. Information we do NOT collect

We do not collect, store, or process:

  • The files you upload or convert
  • The content of your files
  • Your name, email address, or any personal identification information
  • Your file names or file metadata

No account or registration is required to use Convify.

3. Information we may collect automatically

Like most websites, we may collect basic, anonymous usage data through third-party analytics and advertising services, including:

  • Pages visited and features used (anonymous)
  • Browser type, operating system, and device type
  • Approximate geographic location (country/city level only, derived from IP address)
  • Referring website or search query

This data is collected in aggregate and cannot be used to identify individual users.

8. Data security
